# Microsoft AutoGen

A comprehensive hands-on repository for learning and implementing **Microsoft AutoGen**, a powerful framework for building intelligent multi-agent AI systems. This project explores agent orchestration, multi-agent collaboration, tool integration, autonomous workflows, and real-world AI automation use cases.

## 🚀 Overview

Microsoft AutoGen is an open-source framework designed for building AI agents that can communicate, collaborate, and solve complex tasks through multi-agent conversations. It enables developers to create scalable agentic applications with support for LLMs, tools, code execution, and human-in-the-loop workflows.

This repository contains:

* AutoGen fundamentals and setup
* Multi-agent conversation patterns
* Assistant and User Proxy agents
* Tool calling and function execution
* Group Chat and Agent Orchestration
* Autonomous workflows
* Human-in-the-loop systems
* Real-world agentic AI use cases
* Best practices for building production-ready AI agents

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/mahesh-sketch/Microsoft-Autogen.git
cd Microsoft-Autogen
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the environment:

### Windows

```bash
venv\Scripts\activate
```

### Linux/Mac

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```
